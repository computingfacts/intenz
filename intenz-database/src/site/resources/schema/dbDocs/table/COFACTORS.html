ORDER_IN NUMBER(3,0) No 1 4 Number to be used to order cofactors.

WEB_VIEW VARCHAR2(50 BYTE) No null 6 (not used: cofactors are shown in INTENZ and ENZYME views)
COMPOUND_ID NUMBER(15,0) Yes null 7 Internal ID of the compound acting as cofactor.
OPERATOR VARCHAR2(5 BYTE) Yes null 8 Operator (AND, OR) applied to cofactors, if more than one per enzyme.
OP_GRP VARCHAR2(10 BYTE) Yes null 9 Operator group to insert the compound into (in complex cofactor setups).
