COLUMN_NAME DATA_TYPE NULLABLE DATA_DEFAULT COLUMN_ID COMMENTS
ENZYME_ID NUMBER(7,0) No null 1 Enzyme_id is used as an internal unique identifier throughout the database as some entries don't have an EC number, e.g. suggested entries. Example is '70001', '9945', etc
NAME VARCHAR2(1500 BYTE) No null 2 Name stores the actual name of an enzyme which could be a systematic name, a common name, or an other/alternate name, e.g. 'alcohol:NAD oxidoreductase', 'alcohol dehydrogenase', 'aldehyde reductase' (taken from 1.1.1.1 - IUBMB).
NAME_CLASS VARCHAR2(3 BYTE) No null 3 There are three types of names or name_classes: common name, systematic name, and other name(s).
WARNING VARCHAR2(3 BYTE) Yes null 4 Some names of an enzyme are ambiguous, misleading, obsolete, which is stored as 'AMB', 'MIS', 'OBS'.
STATUS VARCHAR2(2 BYTE) No null 5 Internal code of status, for example 'NO', 'UN', 'SN', 'OK', etc.
SOURCE VARCHAR2(6 BYTE) No null 6 Database internal code, for example 'INTENZ', 'IUBMB', 'BRENDA', 'SIB' etc. Each enzyme belongs to one of them.
NOTE VARCHAR2(200 BYTE) Yes null 7 Ambiguous, misleading, obsolete names have sometimes a comment attached to them which can be stored in this column.
ORDER_IN NUMBER(4,0) No null 8 Order applied to names for the same enzyme.
WEB_VIEW VARCHAR2(50 BYTE) No null 9 View to display the name in.
